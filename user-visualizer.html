Music Visualizer

For this project we have used the data set from the Spotify API which describes different variables of a song such as the tempo, loudness, energy, and emotion. We have used these variables and connect it with a visual attribute.

  • Tempo - pulsing of the particles
  • Loudness - amount of particles
  • Energy - the speed of the movements
  • Emotion - color of the particles

On the comparison page you can compare your favorite song with ours!
